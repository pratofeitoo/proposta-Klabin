# Klabin Proposal - DiverCidade

This repository contains the structured proposal for Klabin.

## Setup for GitHub Pages

1.  **Images**: Ensure the `img/` directory contains:
    *   `logo-light.png` (DiverCidade logo with dark text)
    *   `logo-dark.png` (DiverCidade logo with white text)
2.  **Deployment**: This project is ready to be deployed via GitHub Pages by pointing to the `index.html`.

## Theme Support

The page supports both Light and Dark modes, persisted via localStorage.
